# MandelbrotShader
An app that renders the mandelbrot set using shaders

## Controls
Key               | Action
:-----            | :-----
left mouse button | Move view position
mouse wheel       | Zoom
F5                | Switch renderer (Vulkan, OpenGL)
F7                | Previous colormap
F8                | Next colormap
PageDown          | Less iterations
PageUp            | More iterations
